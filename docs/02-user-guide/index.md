# 用户指南概览

本目录是 `sinan-captcha` 的公开使用文档，只回答“怎么使用项目”这一件事。

项目当前对外口径已经固定为双 CLI：

- `sinan-generator`：Go 生成器
- `sinan`：Python 训练与数据工程 CLI

本目录建议按下面顺序阅读：

1. [用户指南总览](./user-guide.md)
2. [使用交付物与正式 CLI](./use-build-artifacts.md)
3. [Windows 训练机安装与模型训练完整指南](./from-base-model-to-training-guide.md)
4. [训练完成后的模型使用与测试](./use-and-test-trained-models.md)
5. [如何确认 Windows 电脑上的 CUDA 版本](./how-to-check-cuda-version.md)

这里不包含：

- 内部设计文档
- 项目阶段文档
- AI 协作规则
- 维护者过程说明

如果你要的是维护仓库、推进实现或查看内部设计，请转到开发者指南和项目开发文档，而不是继续停留在这里。
