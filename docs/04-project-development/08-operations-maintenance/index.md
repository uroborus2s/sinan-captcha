# 运维与维护概览

本目录收纳部署、运行、监控、故障处理和支持手册。

- 本页是该目录的正文首页，用于说明范围、读者和维护边界。
- 目录树、页面路径和访问级别统一由根 `docs/index.md` 声明，这里不重复维护页面清单。
- 本目录下的 Markdown 页面、契约文件和资源文件应随内容变更一起演进。

当前运维与维护文档围绕两类对象组织：

- 训练机与生成器运行维护
- 目标 solver 交付与部署维护

1. [部署说明](./deployment-guide.md)
2. [维护运行手册](./operations-runbook.md)
