## 1. 使用方式

这份文档是进入实现前的执行表，不是泛泛的任务清单。

执行原则：

1. 严格按 `TASK-GEN-001` 到 `TASK-GEN-012` 顺序推进。
2. 每个任务都要产出可检查物，不接受“口头完成”。
3. 前一任务未通过验收，后一任务不得启动。
4. 如果执行中发现要修改样本契约、素材规范或目录骨架，必须回到对应任务重开，不允许在实现中偷偷改。
